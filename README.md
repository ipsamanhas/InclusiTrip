# InclusiTrip
InclusiTrip is a platform where travellers can enter their accessibility needs and be matched with travel services such as hotels, in their destination of choice. This assists in bridging the accessibility gap that already exists in the tourism industry.
It is designed for travellers with diverse accessibility needs, including individuals with mobility challenges, sensory sensitivities, cognitive or neurodivergent conditions, and temporary or situational challenges like any sorts of injuries or fatigue. Additionally, caregivers, family members, and travel planners who assist these individuals can also benefit from this platform.

# Getting Started

### 1. Clone the repository

Open **Terminal** (macOS/Linux) or **Command Prompt** / **PowerShell** (Windows) and run:

```bash
git clone <your-repo-url>
cd <project-directory>
```

### 2. Create a virtual environment

**macOS / Linux (Terminal):**

```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows (Command Prompt):**

```cmd
python -m venv venv
venv\Scripts\activate
```

**Windows (PowerShell):**

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
```

> If PowerShell blocks the activation script, run `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned` once, then try again.

### 3. Install dependencies

In the same terminal (with the virtual environment activated), run:

```bash
pip install -r requirements.txt
```

### 4. Run the development server

```bash
fastapi dev
```

The site will be available at [http://127.0.0.1:8000](http://127.0.0.1:8000).

To stop the server, press `Ctrl+C` in the terminal. When you're done working, deactivate the virtual environment with `deactivate`.
